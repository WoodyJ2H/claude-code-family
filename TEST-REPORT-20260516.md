## Issues Fixed During Testing

### Issue 1: Drive Letter T: Not Mapped
**Problem:** Script hardcoded `T:\` as source, but when running from UNC path (`\\ROGSTRIXJH\TwinSetup\INSTALL.ps1`), the drive letter T: doesn't exist in that PowerShell context.

**Solution:** Modified script to auto-detect T: mapping; if not found, falls back to UNC path `\\ROGSTRIXJH\TwinSetup\`

```powershell
$twin = if (Test-Path "T:\") { "T:\" } else { "\\ROGSTRIXJH\TwinSetup\" }
```

**Result:** Script now works with or without pre-mapped drive letter.

### Issue 2: PowerShell Non-Interactive Mode
**Problem:** Final `pause` command fails because PowerShell runs in non-interactive mode.

**Impact:** None — all copy/install operations complete before the `pause` is reached.

**Solution:** Safe to ignore; acceptable behavior for automated execution.
